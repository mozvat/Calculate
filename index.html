Calculate

Basic example of finding a median from an array of numbers.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the wonderful code that discovers the median from an array of numbers.

Read the definition of the median here: Wiki: Median-Definition

Business rules for the Median calculation

1. Create an array of numbers

2. Sort the array's values in ascending order

3. Determine the middle array item

* If, there is an odd number of array items, then simply get the middle item within the array and it's value is the median.

* If, there is an even number of array items, then find the middle two items within the array, add their values together, then divide by two to find the median.

4. Return the value of the middle array item

Software Design

I applied basic Single Responsibility principle ( Wiki: Single Responsibility Definition )design by modulating the calculation logic from the 'Main' Console posing as a User-Interface.
